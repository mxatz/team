Detailed Description

Implementation of Fifo's functions using c.

The functions that were implemented are MyFIFOInit(): inits the module MyFIFOInsert(): Adds an element to the FIFO MyFIFORemove(): Removes an element from the FIFO MyFIFOPeep(): Returns the oldest element on the FIFO, but does not remove it MyFIFOSize(): Returns the number of elements on the FIFO
